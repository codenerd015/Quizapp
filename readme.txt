Titel: Quiz app 
beschrijving: de quiz app test uw algemene kennis.
 
Functionaliteiten:  de app biedt je keuze menu aan categorie vragen. 
                    de vragen worden opgehaald via API zie link. 
                    de app toont (x aantal) vragen en 4 verschillende antwoord opties.  
                    automatisch berekenen van de score op basis van juiste antwoorden. 
                    feedback per vraag, direct indicatie of een antwoord goed of fout is. 
                    eindresultaat, overzicht van de behaalde score.
		            voor  elk vraag is er 20 sec tijd om te beantwoorden. 


Technologieen: HTML, CSS en JS, Visual code 

API: https://opentdb.com/api_config.php
